# Domain App Tokens

These tokens are shared across all domain apps (vendors, stocks, admin, etc.).
Apps that need a different design system import `@haderach/shared-ui/theme` and
define their own tokens instead.

Defined in `@haderach/shared-ui/theme/domain` (`theme/domain.css`).
