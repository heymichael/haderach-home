Arcade Business Platform

What is Arcade Business Platform?

Arcade Business Platform is built for managing business processes in an AI-first world. Each business domain—vendors, commodities, and more to come—gets a dedicated workspace with an AI agent, interactive dashboards, structured data views, and domain-specific modules tailored to that area of the business. The agent isn't bolted on; it's the primary interface, with traditional views and specialized tools alongside it for when you need them.

Views

Every domain includes these three core views. Individual domains may also have additional views specific to their data.

Agent

A conversational AI assistant built into every domain. Ask questions in plain English, get instant answers, and make changes through chat. The agent understands the domain's data and can read, search, analyze, and edit records on your behalf.

Dashboards

Traditional dashboard views with charts and summary metrics. Filterable by relevant dimensions for each domain. Dashboards are not controlled by the agent—they are standalone, visual interfaces for at-a-glance insights.

Database views

Searchable, sortable tables of your data with inline editing. Database views can be controlled by the agent: when you ask the agent to edit a record, the confirmation modal and edit forms operate on the same underlying data. Use the database view for browsing and manual edits, or pair it side-by-side with the agent to make changes conversationally.

Layout & Navigation

Arcade Business Platform is organized around domains in the left rail. Each domain has its own AI agent and a set of views specific to that domain. The top bar controls what you see in the main content area.

Domains

The left rail lists your available domains (e.g. Vendors, Commodities). Selecting a domain loads its agent and views. Each domain is a self-contained workspace—the agent understands that domain's data and the views are purpose-built for it.

How the top bar works

The top bar lets you toggle between the agent, views, or both. There are four display modes:

1

Full-screen agent

Select Chat in the top bar, deselect everything else. The agent takes the full screen.

2

Full-screen view

Select a view pane, deselect the agent. Dashboard or database fills the screen.

3

Agent + one view

Select Chat plus one view. Side-by-side: agent on the right, view on the left.

4

Switch the view

Select a different view while the agent stays open. The view swaps, the chat persists.

Vendor Management

A complete vendor management domain with an AI agent, spend analytics, and a vendor database—all in one workspace.

The Agent

The centerpiece of Arcade Business Platform. A conversational AI assistant that understands your vendor data. Ask questions in plain English and get instant answers, analytics, and edits—all with enterprise-grade access controls.

The sections below cover everything the vendor agent can do in detail.

Permissions & Data Access

Every agent request is authenticated with your Firebase identity. Spend data visibility is enforced server-side—access controls are applied before any results are generated, so the AI model only ever sees data you are authorized to access.

How access is determined

When you ask the agent about spend, the system resolves your identity and computes exactly which vendors you can see. This happens on every request. The database query itself is scoped to your allowed set, and only the pre-filtered, aggregated results are passed to the AI model so it can compose a natural language answer. Large exports (CSVs) bypass the model entirely and are sent straight to your browser.

1

Authenticate

Your Firebase ID token is verified on every request. The agent resolves your user record and role assignments from the database.

2

Check for Finance Admin

Users with the finance_admin role bypass all filtering and see spend data across every vendor. If you have this role, you're done—full access.

3

Resolve your vendor set

For non-admin users, the system computes your effective vendor list using three inputs:

Access Control What it does Example
Allowed Departments Grants access to all vendors in the specified departments Allowed departments: Engineering, Product → see all vendors assigned to those departments
Allowed Vendors Adds individual vendors to your access set, regardless of department Even if you only have Engineering, you can be granted access to a specific Marketing vendor
Denied Vendors Removes specific vendors from your access set, overriding the above You have Engineering access but a sensitive vendor in that department is excluded
The formula: Your effective access = (Vendors in Allowed Departments ∪ Allowed Vendor IDs) − Denied Vendor IDs. This is computed server-side on every request. Database queries are scoped to your allowed vendor set, and the AI model only sees the resulting aggregates—never raw data outside your access.
AI data usage is under your control. The agent sends pre-filtered, aggregated results to the AI model (currently OpenAI) so it can compose a natural language response. How the AI provider handles that data—including training, retention, and logging—is governed by the settings in your AI platform account. You are responsible for reviewing and configuring those policies to match your organization's data handling requirements.

Hidden vendors

Admins can mark vendors as hidden from agent. These vendors are excluded from all agent search, list, and analytics results for every user, regardless of role. This is useful for internal or test vendors that shouldn't appear in chat conversations.

Vendor Spend Analytics

Ask spending questions in plain English. The agent translates your question into a precise database query, respects your access controls, and returns the answer instantly.

What you can ask

  • $
    Total spend — Grand totals for any time period, with optional filters by department, payment method, owner, and more.
  • #
    Spend by vendor — Monthly breakdown for one vendor, or ranked list of all vendors by spend amount.
  • ÷
    Spend by dimension — Group spend by department, payment method, account type, billing frequency, source system, owner, or 1099 status.
  • ↑
    Top N vendors — Ranked list of your highest-spend vendors with custom N and filters.

Vendor Search & Lookup

Find vendors by name, count them by category, or pull filtered lists. The agent uses fuzzy matching so you don't need to know exact names.

Smart name resolution

The agent uses fuzzy matching to find vendors even when you use partial names, abbreviations, or slight misspellings. If multiple vendors match, it will ask you to pick the right one.

Scenario — Fuzzy matching & disambiguation
You
"Look up Datadog"
H

You
"Look up Acme"
H
I found multiple matches. Did you mean:
• Acme Corp
• Acme Services LLC
• Acme Digital Solutions

Listing & counting

Try saying"List all ACH vendors"
Try saying"How many 1099 vendors?"
Try saying"Count by payment method"
Try saying"Vendors in Engineering"

CSV downloads for large lists

When a list returns more than 10 vendors, the agent will mention that a downloadable CSV is available. A download button appears in the chat so you can export the full list to a spreadsheet.

Typo correction: If you misspell a filter value (e.g., "Mrketing" instead of "Marketing"), the agent will suggest the correct value and ask you to confirm.

Smart Vendor Editing

Edit vendor attributes through natural conversation. Every change goes through a confirmation step—the agent never writes data silently.

How edits work

1

Ask for the change

Tell the agent what you want to update in plain language.

2

Review the proposal

A confirmation modal shows exactly what will change, with current vs. new values.

3

Confirm or cancel

Click confirm to apply, or cancel to discard. Nothing is saved until you confirm.

What you can edit

Department, owner, secondary owner, billing frequency, account type, and purpose. For additional fields, say "edit" followed by the vendor name to open the full edit form.

Bulk CSV Editing

Need to update many vendors at once? The agent generates a pre-filled CSV that you can edit in any spreadsheet tool, then upload back for a batch update with a full diff review.

The workflow

1

Request a CSV

Ask for a vendor export, optionally filtered by department or owner.

2

Download & edit

Click the download button. Edit cells in Excel, Sheets, or any CSV editor.

3

Upload the CSV

Attach your edited CSV using the paperclip icon in the chat.

4

Review & confirm

A diff view shows every change per vendor. Confirm the batch or cancel.

Help Us Improve

Every agent response has thumbs up / thumbs down buttons. Your feedback is stored per-message and helps us identify where the agent is doing well and where it needs improvement.

↑

Thumbs up

The response was accurate and helpful. This reinforces good behavior.

↓

Thumbs down

Something was wrong or unhelpful. You can optionally add a comment explaining what went wrong.

Your feedback matters. We review flagged responses regularly to tune the agent's behavior, fix edge cases, and improve accuracy. Even a simple thumbs-down without a comment is valuable.
